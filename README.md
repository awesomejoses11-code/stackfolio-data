# Stackfolio

**Stackfolio** - Lightweight client-side tools for portfolio data and quick web utilities. No backend, runs entirely in the browser.

🔗 **Live Demo**: [stackfolio-data.vercel.app](https://stackfolio-data.vercel.app)

### 🧰 Tools Included
| Tool | Description |
| --- | --- |
| **index.html** | Main portfolio/data hub |
| **Neon_Fury.html** | Neon-styled web utility |
| **clipboard assistant.html** | Quick clipboard helper |
| **qr-code-generator.html** | Generate QR codes client-side |
| **preview.html** | Live preview utility |

All tools run 100% in the browser. No installs, no API keys, no backend.
